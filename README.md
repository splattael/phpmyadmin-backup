# Phpmyadmin::Backup

Backup SQL dumps via phpmyadmin

## Installation

Add this line to your application's Gemfile:

    gem 'phpmyadmin-backup'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install phpmyadmin-backup

## Usage

    $ phpmyadmin_backup URL USERNAME PASSWORD
